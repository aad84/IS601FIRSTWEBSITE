Linux Commands

1.ls = directory listing\n 2.ls -al = formatted listing with hidden files\n 3.cd dir = change directory to dir 4.cd = change to home 5.pwd = show current directory 6.mkdir dir = create a directory dir 7.rm file = delete file 8.rm -r dir = delete directory dir 9.ps = display your currently active processes 10.top = display all running processes

Vi Commands

1.vi filename = edit filename starting at line 1 2.:x = quit vi, writing out modified file to file named in original invocation 3:wq = quit vi, writing out modified file to file named in original invocation 4.:q = quit (or exit) vi 5.i = insert text before cursor, until hit 6.I = insert text at beginning of current line, until hit 7.a = append text after cursor, until hit 8.A = append text to end of current line, until hit 9.r = replace single character under cursor (no needed) 10.x = delete single character under cursor

Git Commands

1.git init = create the repository 2.git add [filename] = add the file(s) to git in preparation for a commit 3.git commit -m “message”= do the commit to save changes 4.git checkout -b branch name = make a new branch and checkout 5.git checkout branch name = checkout branch 6.git merge BRANCHNAME = merge the branch into current one 7.git status = show modified files in working directory, staged for your next commit 8.git log = show all commits in the current branch’s history 9.git rm [file] = delete the file from project and stage the removal for commit 10.git clone [url] = retrieve an entire repository from a hosted location via URL
